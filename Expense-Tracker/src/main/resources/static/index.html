What's included

Everything you need to stay on budget

Built with Spring Boot + JWT security. Simple, fast, and yours.

JWT Security
Every request is authenticated with a secure JWT token. Role-based access keeps ADMIN and USER data separate.
Dashboard Charts
Category breakdown, monthly trends, daily spending — all your expense data visualized in one place.
Recurring Expenses
Set up Netflix, rent, or any subscription once. It auto-adds every day, week, month, or year.
Date Range Search
Filter expenses between any two dates. Get the total spent with one API call.
Profile Management
Update your name, email, phone, and profile photo. Everything is yours to control.
Forgot Password
Lost your password? Get a secure reset link delivered to your Gmail in seconds.
Dark / Light Mode
Toggle between dark and light theme across every page. Your preference is remembered.
Role-based Access
ADMIN sees all users' expenses. USER sees only their own. Clean separation, no data leaks.
Exception Handling
Every error returns a clean JSON response with status code, message, timestamp, and path.
